## Learnings
- Lua quality tools (stylua, luacheck, busted/luacov) are not installed globally in the environment. The stop hook must verify their existence and instruct the user to install them or gracefully fail, ensuring quality is strictly enforced.
- A doc sync check uses `git diff HEAD --name-only` to ensure documentation matches code changes.
- OpenCode/ClaudeCode Stop hooks are of type "command" and block execution if the script exits with code 2.
